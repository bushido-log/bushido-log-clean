# BUSHIDO LOG（ブシログ）

サムライキングと話しながら、中毒・サボり癖・自己否定を少しずつ直していくためのセルフコーチングアプリです。

---

## これは何？

BUSHIDO LOG は、「サムライキング」という AI コーチと対話しながら、  
毎日のメンタルと行動を整えていくための習慣づくりツールです。

- ムラムラ・不安・落ち込み・サボりたい気分になったときにボタンひとつで呼び出せる
- その瞬間の気持ちに合わせて、短く深いメッセージと「今日やる一歩」を返してくれる
- ログを残していくことで、「自分の物語」を少しずつアップデートしていく

「気合い」とか「根性論」ではなく、  
**問いかけ・気づき・小さな行動** を積み重ねていくのがコンセプトです。

---

## 主な機能

- **サムライキングチャット**
  - 今の気持ちや状況を送ると、サムライキングが落ち着いた口調で返事をくれる
  - 毎回かならず「▶︎ 今日やること」を1つだけ提案

- **今日の目標（デイリーターゲット）**
  - その日に大事にしたいテーマをひと言で決めて、サムライキングと共有
  - 1日を通して「今日の自分の軸」がブレにくくなる

- **振り返りログ**
  - 1日の終わりに、気づき・感謝・反省などをメモとして残せる
  - 後から読み返すことで、自分の成長ストーリーが見えてくる

- **サウンド演出**
  - ボタンタップ時に「太鼓」や「マイクタップ」の効果音が鳴り、少しゲームっぽい体験に
  - 押した瞬間に「よし、やるか」と気持ちを切り替えやすくするための演出

- **シンプル構成**
  - フロントエンド：React Native（Expo）
  - サーバー：Node.js（ローカルで OpenAI API を叩くシンプルな API サーバ）
  - API キーは `.env` に保存し、GitHub にはアップしない構成
